**🧮 Calculator Web App** <br>
A modern and responsive Calculator Web Application built using HTML and CSS with inline JavaScript. It performs basic arithmetic operations with a clean user interface, smooth button interactions, and real-time display updates. <br> <br>
🚀 Features <br>
✅ Perform basic arithmetic operations (+, −, ×, ÷) <br>
✅ Real-time calculation display <br>
✅ Delete (DE) and All Clear (AC) functionality <br>
✅ Responsive design (works on mobile & desktop) <br>
✅ Smooth button hover and click effects <br>
✅ Clean and modern UI <br> <br>
🛠️ Technologies Used <br>
👉🏻 HTML5 – Structure & Logic (inline JavaScript) <br>
👉🏻 CSS3 – Styling & Layout <br> <br>
📂 Project Structure <br>
calculator-web-app/ <br>
│── index.html <br>
│── style.css <br> <br>
⚙️ How It Works <br>
1️⃣ User clicks number and operator buttons <br>
2️⃣ The input is displayed in real-time <br>
3️⃣ On pressing "=" the expression is evaluated <br>
4️⃣ Result is displayed instantly <br> <br>
